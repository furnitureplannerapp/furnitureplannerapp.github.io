Politique de confidentialité

Furniture Planner stocke les projets localement sur votre appareil. Certaines fonctionnalités peuvent utiliser iCloud ainsi que le diagnostic, l’analyse et la configuration distante Firebase. Cette politique explique les données traitées et le fonctionnement du support, des abonnements et des fonctions cloud.

English Polski Deutsch Español Français

Données stockées dans l’app

Furniture Planner peut stocker sur votre appareil les projets, dimensions de meubles, matériaux, accessoires, estimations, paramètres d’export PDF/CSV, noms et coordonnées de clients, textures importées, réglages de l’app et état de l’onboarding.

Si vous choisissez d’utiliser la synchronisation ou sauvegarde iCloud/CloudKit, les données compatibles peuvent être stockées dans votre compte Apple iCloud pour synchroniser vos appareils. Les données iCloud sont gérées par les services Apple et les réglages de votre Apple ID.

Photos, fichiers, caméra et AR

L’app peut importer des textures de panneaux ou images de projet depuis Photos ou Fichiers lorsque vous les sélectionnez. La caméra sert à l’aperçu AR et à la visualisation à l’échelle réelle.

Diagnostic et support

Lorsque vous contactez le support, vous décidez si vous envoyez un rapport et si vous joignez des diagnostics. Un rapport peut inclure un JSON de projet, une capture, la version de l’app, la version iOS, le modèle d’appareil, l’ID projet et les logs récents. Ces données servent uniquement à diagnostiquer le problème signalé.

Diagnostic, analyse et configuration distante Firebase

Furniture Planner utilise des services Firebase fournis par Google LLC ou Google Ireland Limited, selon votre juridiction. Les données Firebase peuvent être traitées conformément à la Politique de confidentialité de Google et aux informations Firebase sur la confidentialité et la sécurité.

Pour le diagnostic, l’analyse et la configuration de l’app, l’app peut collecter ou envoyer des données techniques telles que le type d’appareil, la version iOS, la version de l’app et le numéro de build, la langue ou la région de l’app, les informations sur les crashs et erreurs, les événements analytiques de base liés à l’utilisation de l’app ainsi que les identifiants techniques utilisés par Firebase, comme les identifiants d’installation Firebase ou les identifiants d’installation Crashlytics.

Firebase Crashlytics

Firebase Crashlytics aide à détecter, analyser et corriger les crashs de l’app. Il peut collecter des rapports de crash, des journaux de diagnostic, des informations sur l’appareil et des informations sur la version de l’app. Ces données sont utilisées uniquement pour améliorer la stabilité de l’app et résoudre les erreurs.

Firebase Analytics

Firebase Analytics nous aide à comprendre de manière générale comment l’app est utilisée, quelles fonctions sont utilisées et où les utilisateurs peuvent rencontrer des problèmes. Nous utilisons l’analyse pour des enseignements agrégés sur le produit et la qualité, et non comme outil d’identification de personnes nommées. Les événements peuvent inclure l’ouverture d’écrans, l’utilisation de fonctions, le début ou la fin d’un parcours d’achat ou d’abonnement, la restauration d’un abonnement, des erreurs de configuration ou l’ouverture de messages de mise à jour.

Firebase Remote Config

Firebase Remote Config télécharge la configuration depuis les serveurs Firebase afin que l’app puisse recevoir des réglages opérationnels sans mise à jour de l’app. Il peut être utilisé pour la version ou le build minimum pris en charge, les messages de service, le mode maintenance, les messages de mise à jour, la disponibilité des fonctions et la configuration des fonctions. Firebase Remote Config n’est pas utilisé pour stocker des projets de meubles privés.

Abonnements

Les abonnements et achats sont traités par Apple via StoreKit et l’App Store. Furniture Planner reçoit les informations nécessaires pour déverrouiller des fonctions, mais les données de paiement sont gérées par Apple.

Pas de publicité ni de vente de données

Furniture Planner ne vend pas vos données et n’intègre pas de SDK publicitaire tiers. Firebase Analytics est utilisé pour le diagnostic de l’app et l’analyse produit décrits ci-dessus.
