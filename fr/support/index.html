Support

Comment pouvons-nous aider ?

Furniture Planner est activement développé. Si un projet, une estimation, un PDF, la 3D ou un import/export pose problème, envoyez-nous un signalement clair.

English Polski Deutsch Español Français

Contacter le support

Écrivez à furnitureplanner.contact@gmail.com. Indiquez une brève description, les étapes de reproduction, le résultat attendu et le résultat actuel.

Pièces jointes utiles

Captures, JSON de diagnostic, version de l’app, version iOS, modèle d’appareil et ID projet nous aident à reproduire plus vite le problème. Vous décidez si vous joignez des données de diagnostic.

Traitement des signalements

Les données transmises dans un signalement au support sont utilisées uniquement pour analyser, reproduire et résoudre le problème signalé.

L’envoi de données de diagnostic est volontaire.

Délai de réponse

Nous faisons de notre mieux pour répondre aux signalements aussi rapidement que possible, mais nous ne garantissons aucun délai de réponse précis.

Périmètre du support

Le support couvre le comportement de l’app, les exports, les diagnostics et la cohérence des données. Les décisions de production doivent toujours être vérifiées avant fabrication.
