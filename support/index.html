Support

How can we help?

Furniture Planner is actively developed. If something looks wrong in a project, estimate, PDF, 3D preview or import/export flow, send us a clear report.

English Polski Deutsch Español Français

Contact support

Email us at furnitureplanner.contact@gmail.com. Include a short description, steps to reproduce the issue, expected result and actual result.

Helpful attachments

Screenshots, diagnostic JSON exports, app version, iOS version, device model and project ID can help us reproduce the issue faster. You decide whether to include diagnostic data.

Report processing

Data submitted in a support report is used only to analyze, reproduce and resolve the reported issue.

Submitting diagnostic data is voluntary.

Response time

We do our best to respond to reports as quickly as possible, but we do not guarantee a specific response time.

Response scope

Support focuses on app behavior, exports, diagnostics and data consistency. Production decisions should always be verified before manufacturing.
