¿Cómo podemos ayudar?

Furniture Planner se desarrolla activamente. Si algo falla en un proyecto, presupuesto, PDF, 3D o importación/exportación, envíanos un reporte claro.

English Polski Deutsch Español Français

Contactar soporte

Escríbenos a furnitureplanner.contact@gmail.com. Incluye una descripción breve, pasos para reproducir, resultado esperado y resultado actual.

Adjuntos útiles

Capturas, JSON diagnóstico, versión de la app, versión de iOS, modelo del dispositivo e ID del proyecto nos ayudan a reproducir el problema más rápido. Tú decides si adjuntas datos de diagnóstico.

Procesamiento de reportes

Los datos enviados en un reporte de soporte se utilizan únicamente para analizar, reproducir y resolver el problema reportado.

El envío de datos de diagnóstico es voluntario.

Tiempo de respuesta

Hacemos todo lo posible por responder a los reportes tan rápido como sea posible, pero no garantizamos un tiempo de respuesta específico.

Alcance del soporte

El soporte cubre comportamiento de la app, exportaciones, diagnóstico y consistencia de datos. Las decisiones de producción siempre deben verificarse antes de fabricar.
